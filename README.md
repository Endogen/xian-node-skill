# xian-node-skill

OpenClaw skill for deploying and managing Xian blockchain nodes.

## Features

- Join mainnet/testnet networks
- Create new Xian networks
- Node monitoring and management
- Validator key generation

## Usage

Install as an OpenClaw skill or read [SKILL.md](SKILL.md) directly.

- [Xian](https://xian.org) — L1 blockchain with native Python smart contracts
- [xian-stack](https://github.com/xian-network/xian-stack) — Docker-based node setup
- [OpenClaw](https://github.com/openclaw/openclaw) — AI agent framework
